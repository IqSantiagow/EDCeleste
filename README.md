**EDCeleste - LLM powered terminal-like Copilot for Elite Dangerous game with voice interface**

App provides seamless experience within the game using STT, TSS and LLM services to sound like human-like pilot

Celeste reacts to the in-game events in realtime and decides to which answer or react in game (by keystrokes or game actions)
It can decide to fetch informations from EDMC, Galnet or other various E:D api resources to get information needed for current context or query from user
